CERAMICO — Website Files
=========================

HOW TO USE:
  1. Copy this entire folder into your project ROOT (same level as /fotebi/)
  2. Open index.html in your browser to start

FILES:
  index.html       - Georgian homepage
  home.html        - English homepage
  koleqcia.html    - Georgian collection
  collection.html  - English collection
  teqnologia.html  - Georgian technology
  Technology.html  - English technology

  style.css        - Shared styles (ALL pages use this)
  koleqcia.css     - Extra styles for koleqcia.html
  collection.css   - Extra styles for collection.html
  teqnologia.css   - Extra styles for teqnologia.html
  Technology.css   - Extra styles for Technology.html

LANGUAGE SWITCHING:
  index.html      ↔ home.html
  koleqcia.html   ↔ collection.html
  teqnologia.html ↔ Technology.html
